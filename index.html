Emily | Amazon Alexa skill for Smarthomes

Emily is a system that is comprised of the necessary components to allow caregivers the ability to learn about their loved ones. For example, someone might ask "Emily, did my grandmother sleep enough last night?" which communicates directly with internal CASAS systems using machine learning on data collected from sensors inside of the home. I am following the work done by two research undergraduates who first explored this space.

Technology Stack: Python, Docker, PostgreSQL, AWS

Flash Volunteer | Web Application for Volunteering

FlashVolunteer was a 501c3 that aimed to make volunteering easier to do by providing an online application that connected volunteers with organizations in their area. This effort was started by several software engineers and technologists who wanted to expand the value of volunteerism. I joined as a volunteer engineer and later joined their board of directors and remained option until the organization disbanded in 2016.

Technology Stack: Postgresql, Sass, Ruby, Rails


EECS Placement Test | Web Application for Class Placement

Students who are looking to study Electrical Engineering, Computer Engineering or Computer Science at WSU are required to take the same entry-level course (CPTS 121). Previously, students were asked if they had any previous programming experience and if they didn't they would be recommended to take an easier course (CPTS 111) which teaches the basics of computer science. I worked as the sole developer under Dr. Sakire Arslan Ay to create an online testing application which better sorts students into the proper course based on a series of questions.

Technology Stack: Postgresql, Sass, Ruby, Rails


pClust | Application Porting to macOS

Following the work of Dr. Svetlana Lockwood, I worked to modify her application that created and analyzed graphs of proteins to work on macOS.

Technology Stack: Java


Sequence Pattern Analyzer | Graphical Interface for DNA Analysis

A graduate student built an algorithm and platform to analyze FASTA files for the USDA-ARS and post-doctorate researchers at WSU. This platform took a handful of patterns (pseudogenes) and matched them against a much larger set of samples. The samples were essentially composed of pieces from the patterns. My work was to create a friendly user interface for researchers to use.

Technology Stack: Python
